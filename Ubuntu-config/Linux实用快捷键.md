# Linux 实用快捷键

1. 移动快捷键

   Ctrl + a     跳到行首
   Ctrl + e     跳到行尾

2. 删除快捷键

   Ctrl + d    向右删除一个字符

   Ctrl + u    删除当前位置字符至行首（输入密码错误的时候多用下这个）
   Ctrl + k    删除当前位置字符至行尾

   Ctrl + w   删除从光标到当前单词开头

3. 复制、粘贴

   Ctrl + Shift + V     粘贴

   Ctrl + Shift + C     复制

   Ctrl + Y   

4. 其他

   Ctrl + l-- 清屏

   Ctrl + c-- 终止操作
